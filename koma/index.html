Features

読書に集中できる
6つの特徴

ライブラリ管理から快適なリーダーまで、マンガを読むために必要な機能を一通り搭載しています。

📁

幅広いフォーマットに対応

ZIP / CBZ / RAR / CBR / EPUB。主要な圧縮・電子書籍形式をすべてサポート。2重圧縮ファイルも対応。

📚

使いやすいライブラリ管理

フォルダブラウジングで端末内のファイルをそのまま整理。読了・未読の管理、サムネイル表示、ソート機能を搭載。

⚡

高速表示

大きなファイルでも最初の数ページをすぐに表示。バックグラウンドで残りを読み込むので待ち時間を最小化。

📖

快適なリーダー

縦スクロール・横スワイプ対応。ピンチズーム・ページスライダー搭載。画面回転の固定もワンタップで切り替え可能。

✅

一括操作

長押しで複数ファイルを選択。読了マーク・未読に戻す・削除をまとめて実行。

🔒

広告は読書後のみ

読んでいる最中は広告ゼロ。リーダーを閉じたあと、数回に1回だけ表示されます。
